Static Assets (No Token Required)

The following assets are served from /docs/static/ without requiring tokens:

  • CSS files: static/main.css (loaded above)
  • JavaScript files: static/app.js (loaded below)
  • Images and other assets in static/ directory

Benefits

  • ✅ Simple HTML - Standard <link> and <script> tags work
  • ✅ Better performance - Assets cached normally by browsers
  • ✅ Easier integration - No token management for CSS/JS
  • ✅ Security maintained - HTML content still requires tokens

How It Works

URLs like /docs/static/main.css are served publicly, while documents like this HTML file still require valid tokens for access.
